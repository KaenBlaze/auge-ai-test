# Sensor Catalog

The AUGE platform supports the following sensor types:

## Temperature Sensors
- RTD (Pt100, Pt1000)
- Thermocouples (Type K, Type J)
- Infrared pyrometers

## Pressure Sensors
- Gauge pressure transducers (0–100 bar)
- Differential pressure sensors
- Absolute pressure sensors

## Vibration Sensors
- Accelerometers (IEPE, MEMS)
- Velocity transducers
- Proximity probes for shaft vibration

## Protocols
- Modbus TCP/RTU
- OPC-UA
- MQTT
- 4–20 mA analog inputs via edge gateways

All sensors must be registered in the AUGE device registry before data ingestion begins.
